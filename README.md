# 2450_IceCubed
 Wind Chill 2450's 2023 Charged Up Code (FRC)

## Robot components:
* Swerve Drive
* Rotating arm
* Pneumatic rock ratcheting system
* Vision (limelight  3)
* Intake mechanism
* FPV camera
* LEDs

## Motor controllers:
 The robot uses Rev Spark Max motor controllers with NEO brushless motors.
